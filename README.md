# 輔大教室課表查詢

這是一個純靜態的教室課表查詢頁面，用來查詢單一教室在一週內各節次的占用情況。使用者輸入完整教室代碼後，可以查看該教室從週一到週六的排課結果；若沒有完全相符的教室，介面會提供最接近的教室代碼建議。

## 專案內容

- 單一教室查詢，支援完整教室代碼比對
- 查無完全相符時，提供相近教室建議
- 桌面版以週課表矩陣顯示週一到週六、各節次的狀態
- 手機版改用依星期切換的卡片檢視，逐日查看各節次內容

## 資料來源與目前查詢邏輯

- 資料檔案為 `fju_day_courses.json`
- 原始來源記錄於資料中的 `source_url`，目前指向 `http://estu.fju.edu.tw/fjucourse/Secondpage.aspx`
- 前端載入 JSON 後，會建立教室索引並直接在瀏覽器中查詢
- 目前實作只使用第一組排課欄位，也就是 `weekday_1`、`period_1`、`room_1`，並搭配 `week_1`
- `*_2` 與 `*_3` 的排課欄位目前不參與查詢與顯示
- 連續節次會依 `period_1` 展開後填入對應時段，例如 `D1-D2`

## 本機執行

這個專案是靜態網站，直接開啟 HTML 檔可能會因為瀏覽器限制而無法讀取 JSON。請在專案目錄啟動本機伺服器：

```bash
python3 -m http.server 4173
```

啟動後開啟 `http://localhost:4173/`。

## 部署說明

此專案只有 HTML、CSS、JavaScript 與 JSON 靜態檔案，很適合部署到 GitHub Pages。將目前目錄內容發布為靜態站點後，即可直接提供查詢頁面，不需要額外的建置流程或後端服務。

## 檔案結構

- `index.html`，頁面結構
- `styles.css`，畫面樣式
- `app.js`，查詢、索引與畫面更新邏輯
- `fju_day_courses.json`，課程資料與中繼資訊
